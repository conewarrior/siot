루미나는 AI를 활용하여 브랜드 아이덴티티에 맞는 디자인 시스템을 자동으로 생성하는 도구입니다.

## 문제

디자인 시스템을 처음부터 구축하는 것은 시간이 많이 걸리고, 일관성을 유지하기 어렵습니다.

## 해결책

브랜드 컬러와 스타일 가이드를 입력하면, AI가 자동으로 컴포넌트, 토큰, 문서를 생성합니다.

## 주요 기능

- 브랜드 컬러 기반 팔레트 자동 생성
- 타이포그래피 스케일 추천
- React 컴포넌트 코드 생성
- Figma 플러그인 연동
