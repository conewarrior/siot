프리즘은 이미지를 분석하여 조화로운 컬러 팔레트를 자동으로 추출하는 도구입니다.

## 기능

- 이미지 드래그 앤 드롭으로 즉시 분석
- 주요 색상 5-10개 자동 추출
- 보색, 유사색 등 컬러 조합 추천
- CSS, Tailwind, Figma 형식으로 내보내기

## 기술적 도전

Canvas API를 사용해 이미지 픽셀 데이터를 분석하고, k-means 클러스터링으로 대표 색상을 추출했습니다.
