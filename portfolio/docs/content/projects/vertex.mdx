버텍스는 브라우저에서 바로 사용할 수 있는 경량 3D 모델링 도구입니다.

## 배경

3D 작업을 위해 무거운 소프트웨어를 설치하는 것은 진입 장벽이 높습니다. 간단한 3D 오브젝트를 만들거나 수정하는 작업을 웹에서 할 수 있다면 어떨까요?

## 주요 기능

- 기본 도형 생성 및 편집
- 재질과 조명 설정
- glTF/GLB 형식 내보내기
- React Three Fiber 컴포넌트 코드 생성

## 결과

디자이너들이 개발자에게 전달할 3D 에셋을 직접 만들고 수정할 수 있게 되었습니다.
