# 라벨링 툴 단축키 UX 개선

## 단축키를 이용한 작업 속도 향상

### 회사
셀렉트스타

### 기간
2022.09 - 2022.11

### 역할
프로덕트 디자이너 (전체 과정)

---

## 비즈니스 임팩트

### 핵심 지표
- 연간 작업비: **4.2억원**
- 전체 매출 대비: **20%**

### OKR 연계
프로젝트 원가 비용 절감 → 작업속도 향상

### 기대 효과
작업속도 **10% 향상** 시 → **4,200만원** 절감 가능

---

## 리서치

### 조사 방법
- **Mission Study**: 사내 스터디그룹
- **User Feedback**: 채널톡 VOC 분석
- **Action Flow 분석**: 작업자 행동 패턴 추적
