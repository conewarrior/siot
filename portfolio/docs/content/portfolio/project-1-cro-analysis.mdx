# CRO 분석 자동화

## 프로젝트 개요

랜딩페이지 전환율 최적화를 위한 데이터 분석 자동화

- **회사**: 지피터스
- **기간**: 2024.12 - 2025.01
- **역할**: UX 리서치, 데이터 분석 (전체 과정)

---

## 문제 정의

기존 CRO 분석 프로세스의 한계:

- **수동 데이터 수집** → 주당 8시간 이상 소요
- **분석 결과 일관성 부재** → 담당자별 다른 기준 적용
- **인사이트 리드타임** → 데이터 수집부터 보고까지 3-5일

{/* Before/After 비교: CROMetricsChart에서 시각화 */}

---

## 해결 과정

### 1. 데이터 파이프라인 구축
- Beusable 히트맵 데이터 자동 수집
- GA4 연동을 통한 전환 데이터 통합

### 2. 분석 자동화
- 패턴 인식 기반 개선점 추천
- 주간 리포트 자동 생성

## 결과

핵심 지표 개선:

- **분석 시간 70% 단축** (주 8시간 → 2.5시간)
- **전환율 15% 향상** (데이터 기반 개선안 적용)
