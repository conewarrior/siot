# 포트폴리오 사이트

## 프로젝트 개요

Next.js 기반의 인터랙티브 포트폴리오 웹사이트

### 역할
- 풀스택 개발
- UI/UX 디자인
- 콘텐츠 기획

### 기간
2024년 12월 - 현재

---

## 문제 정의

포트폴리오 제작의 고민:

- 정적인 PDF 형식의 한계
- 작업물을 효과적으로 보여주는 방법
- 유지보수가 쉬운 구조 필요

---

## 해결 과정

1. **기술 스택 선정**
   - Next.js 16 + App Router
   - MDX 기반 콘텐츠 관리
   - Tailwind CSS v4

2. **인터랙티브 요소**
   - 스크롤 기반 애니메이션
   - 다크모드 지원
   - 반응형 레이아웃

3. **콘텐츠 시스템**
   - MDX로 블로그 및 프로젝트 관리
   - 자동 이미지 CDN 업로드

---

## 결과

- 모던한 포트폴리오 사이트 구축
- 블로그를 통한 지식 공유
- 지속적인 콘텐츠 업데이트 가능

---
