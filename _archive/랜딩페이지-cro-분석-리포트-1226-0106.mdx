### 핵심 발견

- **가격 정보 노출 양호**: 500-1,000px 구간에 가격 정보가 위치하여 85% 도달률 확보

## 6) 섹션별 분석 및 개선안

### 6-1. 히어로 섹션 (0-1,000px)

#### 현황
- 그라데이션 배경(흰색→주황색)에 중앙 정렬 레이아웃
- 메인 헤드라인: "혼자 하면 어려운 AI, 함께하면 쉬워져요! 지피터스 AI스터디 20기"
- 서브 카피: "바이브코딩부터 AI 자동화까지, 역대 최대 대기자..."
- CTA 버튼: "스터디 신청하기 ▶" (주황색)

#### 핵심 메시지
"AI 학습을 혼자 하기 어려운 사람들을 위한 20기 스터디 모집, 지금이 가장 좋은 혜택"

#### 잘 되고 있는 점
✅ 메인 CTA 클릭률 23.2%로 매우 우수
✅ 감성적 카피("혼자 하면 어려운...")가 타겟 페인 포인트를 정확히 타격
✅ 그라데이션 배경이 시선을 CTA로 유도

#### 개선 기회

| 항목 | 내용 |
|------|------|
| **문제** | Average Fold 880px로 가격 정보가 첫 화면에 미노출. 사용자가 스크롤 없이 "얼마인지" 판단 불가 |
| **가설** | 가격 정보를 히어로 섹션 내에 노출하면 신뢰도가 높아지고 전환율이 증가할 것 |
| **개선안** | 히어로 CTA 하단에 "한정 특가: 269,000원 (10% 할인)" 텍스트를 작게 추가하여 가격 anchor 제공 |
| **예상 효과** | 전환율 +3-5%p, 이탈률 -5%p |

| 항목 | 내용 |
|------|------|
| **문제** | CTA 버튼 텍스트가 평범함 ("스터디 신청하기"). 긴급성/한정성 부각 부족 |
| **가설** | 긴급성을 강조한 CTA는 FOMO를 자극하여 즉시 전환을 유도할 것 |
| **개선안** | CTA 텍스트를 "오늘까지 10% 할인 받기 ▶" 또는 "지금 바로 신청하기 (마감 임박)" 로 변경 + 버튼 위에 "⏰ 얼리버드 D-3" 뱃지 추가 |
| **예상 효과** | 클릭률 +2-4%p, 전환율 +1-2%p |

### 6-2. 가격 섹션 (500-1,000px)

#### 현황
- 3단계 가격 카드 (슈퍼얼리버드/얼리버드/정가)
- 그라데이션 카드 디자인 (빨강-노랑, 파랑-분홍, 민트-초록, 회색)
- 할인율 뱃지 (10%, 5%, 0%)

#### 핵심 메시지
"지금 구매할수록 더 저렴하다"

#### 잘 되고 있는 점
✅ 시각적으로 눈에 띄는 그라데이션 카드
✅ 단계별 가격 비교가 명확하여 긴급성 부각

#### 개선 기회

| 항목 | 내용 |
|------|------|
| **문제** | 가격 카드가 정적이며, 현재 진행 중인 단계가 불명확. 사용자가 "지금 어떤 가격이 적용되는지" 혼란 가능 |
| **가설** | 현재 적용 중인 가격을 명확히 표시하면 의사결정 속도가 빨라질 것 |
| **개선안** | 현재 진행 중인 가격 카드에 "✨ 지금 구매 가능!" 또는 "🔥 현재 가격" 뱃지 추가 + 테두리 강조 (펄스 애니메이션) |
| **예상 효과** | 전환율 +2-3%p, CTA 클릭률 +1-2%p |

| 항목 | 내용 |
|------|------|
| **문제** | 가격 카드에 CTA 버튼이 없어 즉시 구매로 연결되지 않음. 클릭 시 스크롤 하단으로 이동 필요 |
| **가설** | 각 가격 카드 내에 구매 버튼을 추가하면 전환 마찰이 줄어들 것 |
| **개선안** | 현재 적용 가격 카드에 "이 가격으로 신청하기" 버튼 추가 (미적용 카드는 비활성화 또는 "종료됨" 표시) |
| **예상 효과** | 전환율 +5-7%p, 가격 섹션 클릭률 +10%p |

## 10) 추가 확인 필요 사항

### 데이터로 확인 안 되는 부분

1. **실제 전환율(CVR)**
   - 뷰저블 데이터에서 #purchase 클릭 후 결제 완료율 불명
   - 필요: GA4 또는 결제 시스템 데이터 연동
   - 예상: 773클릭 중 실제 구매가 몇 건인지 확인 필요

2. **이탈 이유**
   - 76.8% 이탈률의 구체적 원인 불명
   - 필요: Exit Survey 또는 Session Recording 분석
   - 가능성: 가격 부담 / 스케줄 불일치 / 정보 부족

3. **모바일 UX**
   - 모바일 26.8% 트래픽의 별도 분석 필요
   - 필요: 모바일 전용 히트맵 및 스크롤 데이터
   - 확인: 모바일에서 CTA 버튼이 터치하기 쉬운지, 스크롤 피로도는 어떤지

4. **유입 소스별 전환율**
   - 어떤 채널에서 온 사용자가 전환율이 높은지 불명
   - 필요: Segment별 전환율 비교 (Direct vs 내부 링크 vs 외부)
   - 가능성: Direct 유입이 전환율 더 높을 것으로 예상

5. **경쟁사 벤치마크**
   - 비슷한 온라인 교육 상품의 CVR 기준치 불명
   - 필요: 업계 평균 데이터 (일반적으로 2-5% 예상)

6. **A/B 테스트 이력**
   - 과거 개선안 테스트 결과 불명
   - 필요: 이전 기수 랜딩페이지와 비교 데이터

7. **구매 결정 시간**
   - 첫 방문 후 구매까지 평균 소요 시간 불명
   - 필요: 코호트 분석 (첫 방문→재방문→구매 경로)

8. **챗봇 활용도**
   - 4위 클릭 요소(8,253px, 146클릭)가 챗봇으로 추정되나 대화 내용 불명
   - 필요: 챗봇 로그 분석 (주요 질문 TOP10)
